1. Document Metadata

These elements provide information about the document itself, such as its title, settings, and links to external resources (like CSS or scripts). They are placed inside the <head> element.

Key Elements: <head>, <title>, <meta>, <link>, <script>, <style> 

<!-- This code goes in the <head> -->

<title>My Page Title</title>

<meta charset="UTF-8">
<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">

<link rel="stylesheet" href="style.css">

<script src="app.js"></script>

(Output is not visible on the page itself.)

This code would:

  • Set the browser tab's title to "My Page Title".
  • Ensure characters (like emoji) display correctly with UTF-8.
  • Optimize the page for mobile devices.
  • Load an external CSS file to style the page.
  • Load an external JavaScript file to add interactivity.
2. Sectioning and Structural Content

These elements organize the document into logical, semantic sections. They define the page's layout, such as the header, main content area, navigation, and footer.

Key Elements: <body>, <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>, <div> 

<header>
  <h1>My Website</h1>
  <nav>
    <a href="/">Home</a> | <a href="/about">About</a>
  </nav>
</header>

<main>
  <article>
    <h2>Blog Post Title</h2>
    <p>This is the main content...</p>
  </article>
</main>

<footer>
  <p>© 2025 My Website</p>
</footer>

3. Text Content and Semantics

These elements structure and add meaning to text. This includes paragraphs, lists, links, and inline elements that emphasize text (like <strong>) or define it (like <abbr>).

Key Elements: <p>, <a>, <strong>, <em>, <ul>, <ol>, <li>, <blockquote>, <code> 

<p>This is a paragraph with 
  <strong>important</strong> text and 
  <em>emphasized</em> text.
</p>

<p>Read more at <a href="https://google.com">Google</a>.</p>

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

This is a paragraph with important text and emphasized text.

Read more at Google.

  • First item
  • Second item
4. Forms and Input

These elements are used to create interactive forms for collecting user input. This includes text fields, buttons, dropdowns, checkboxes, and more.

Key Elements: <form>, <input>, <label>, <button>, <select>, <option>, <textarea>, <fieldset> 

<form action="/submit">
  <label for="name">Name:</label>
  <input type="text" id="name" name="user_name">

  <label for="job">Job:</label>
  <select id="job" name="user_job">
    <option value="dev">Developer</option>
    <option value="des">Designer</option>
  </select>
  
  <button type="submit">Submit</button>
</form>
5. Media and Embedding Elements

This category includes elements for embedding various types of media, such as images, audio, video, or even other web pages (using <iframe>).

Key Elements: <img>, <audio>, <video>, <iframe>, <picture>, <source>, <canvas> 

<!-- Image with alternative text -->
<img 
  src="https://via.placeholder.com/150" 
  alt="A 150x150 placeholder image"
  width="150"
  height="150"
>

<!-- Basic video embed -->
<video controls width="250">
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
(Video player example)

6. Table Content

These elements are specifically designed for creating and structuring tabular data (data organized in rows and columns).

Key Elements: <table>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>, <caption> 

<table>
  <caption>Monthly Sales</caption>
  <thead>
    <tr>
      <th>Month</th>
      <th>Sales</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>$1,000</td>
    </tr>
  </tbody>
</table>

These elements create user-interactive widgets or are part of the Web Components specification for creating reusable custom elements.

Key Elements: <details>, <summary>, <dialog>, <slot>, <template>

8. Deprecated Elements

These elements are obsolete and no longer recommended. They have been replaced by newer HTML elements or, more commonly, by CSS for styling.

Key Elements: <font>, <center>, <marquee>, <strike> 

<!-- This is BAD PRACTICE. Use CSS instead. -->
<center>
  <font color="red" size="5">
    This text is centered and red.
  </font>
</center>
This text is centered and red.

(Note: Using <font> and <center> is deprecated. Use CSS instead.)

9. Global Attributes

These attributes can be used on any HTML element. They provide core functionality like styling, identification, accessibility, and interactivity.

Key Attributes: id, class, style, title, hidden, lang, tabindex 

<p 
  id="intro-paragraph"
  class="highlight"
  style="color: blue; font-weight: bold;"
  title="This is a tooltip"
>
  This paragraph has global attributes.
</p>
